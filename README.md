# 💄 Common Cosmetics

A social media platform built with Ruby on Rails where users can post photos, comment, like, and share their favorite cosmetic looks and products.

---

## ✨ Features

- 📸 **Posts with Images** — Share your look with a caption and rating
- 💬 **Nested Comments & Replies** — Start threads and engage
- ❤️ **Likes** — Like posts, comments, and replies
- 🧑‍🤝‍🧑 **Followers** — Follow users and see their updates
- 👤 **Public Profiles** — View user bios, avatars, and post history
- ⚡ **Turbo Streams** — Real-time updates with Hotwire
- 🌗 **Dark Mode Toggle** — Red & black or Red & white theme
- 🔐 **Devise Authentication** — Secure sign up, login, and settings
- 🔗 **@Mentions Support** — Mention users in posts and comments with live profile linking
- 🧠 **Autocomplete for Mentions** — Autocomplete usernames as you type with dropdown selector
- 🔔 **Notifications** — Get notified when you're mentioned
- 🧩 **Mention Tracking** — Mentions are stored in the database and deduplicated

---

## 📦 Tech Stack

- **Backend**: Ruby on Rails 7
- **Frontend**: Turbo + Stimulus (Hotwire), Tailwind CSS
- **Database**: PostgreSQL
- **Image Uploads**: Active Storage
- **Authentication**: Devise
- **Realtime UI**: Turbo Frames & Turbo Streams
- **Clean URLs**: FriendlyId

---

## 🚀 Getting Started

### Requirements

- Ruby 3.x
- Rails 7.x
- PostgreSQL

### Setup

```bash
git clone https://github.com/samiesaheb/Common-Cosmetics.git
cd common_cosmetics
bundle install
bin/setup
rails server
```
